sunirīkṣitaniścalakaravallabhadhārājalokṣitā na tathā
sotkampena mayā sakhi dṛṣṭā sā mādyati sma yathā
sakhi moghīkṛtamadane pativrate kas tavādaraṃ kurute
nāśrauṣīr bhagavān api sa kāmaviddho haraḥ pūjyaḥ
sā mayi na dāsabuddhir na ratir nāpi trapā na viśvāsaḥ
hanta nirīkṣya navoḍhāṃ manye vayam apiryā jātāḥ
sucirāyāte gṛhiṇī niśi bhuktā dinamukhe vidagdheyam
dhavalanakhāṅkaṃ nijavapur akuṅkumārdraṃ na darśayati
stanajaghanorupraṇayī gāḍhaṃ lagno niveśitasnehaḥ
priya kālapariṇatir iyaṃ virajyase yan nakhāṅka iva
sā vicchāyā niśi niśi sutanur bahutuhinaśītale talpe
jvalati tvadīyavirahād oṣadhir iva himavataḥ pṛṣṭhe
sā nīrase tava hṛdi praviśati niryāti na labhate sthairyam
sundara sakhī divasakarabimbe tuhināṃśurekheva
sukumāratvaṃ kāntir nitāntasāratvam āntarāś ca guṇāḥ
kiṃ nāma nendulekhe śaśagraheṇaiva tava kathitam
saurabhyamātramanasām āstāṃ malayadrumasya na viśeṣaḥ
dharmārthināṃ tathāpi sa mṛgyaḥ pūjārtham aśvatthaḥ
saṃvāhayati śayānaṃ yathopavījayati gṛhapatiṃ gṛhiṇī
gṛhavṛtivivaraniveśitadṛśas tathāśvāsanaṃ yūnaḥ
satyaṃ svalpaguṇeṣu stabdhā sadṛśe punar bhujaṅge sā
arpitakoṭiḥ praṇamati sundara haracāpayaṣṭir iva
sarvaṃsahāṃ mahīm iva vidhāya tāṃ bāṣpavāribhiḥ pūrṇām
bhavanāntaramayam adhunā saṃkrāntas te guruḥ premā
sambhavati na khalu rakṣā sarasānāṃ prakṛticapalacaritānām
anubhavati haraśirasy api bhujaṅgapariśīlanaṃ gaṅgā
sulabheṣu kamalakesaraketakamākandakundakusumeṣu
vāñchati manorathāndhā madhupī smaradhanuṣi guṇībhāvam
sā lajjitā sapatnī kupitā bhītaḥ priyaḥ sakhī sukhitā
bālāyāḥ pīḍāyāṃ nidānite jāgare vaidyaiḥ
sucirāgatasya saṃvāhanacchalenāṅgam aṅgam āliṅgya
puṣyati ca mānacarcāṃ gṛhiṇī saphalayait cotkalikām
sā sarvathaiva raktā rāgaṃ guñjeva na tu mukhe vahati
vacanapaṭos tava rāgaḥ kevalam āsye śukasyeva
sāyaṃ kāntabhujāntarapatitā ratinītasakalarajanīkā
uṣasi dadatī pradīpaṃ sakhībhir upahasyate bālā
sā tīkṣṇamānadahanā mahataḥ snehasya durlabhaḥ pākaḥ
tvāṃ darvīm iva dūti prayāsayannasmi viśvastaḥ
snehakṣatir jigīṣā samaraḥ prāṇavyayāvadhiḥ kariṇām
na vitanute kam anarthaṃ dantini tava yauvanodbhaṭaḥ
sadanād apaiti dayito hasati sakhī viśati gharaṇim iva bālā
jvalati sapatnī kīre jalpati mugdhe prasīdati
saṃkucitāṅgīṃ dviguṇāṃśukāṃ manomātravisphuranmadanām
dayitāṃ bhajāmi mugdhām iva tuhina tava prasādena
sakhi lagnaiva vasantī sadāśaye mahati rasamaye tasya
bāḍavaśikheva sindhor na manāg apy ārdratāṃ bhajasi
sakhi mihirodgamanādipramodam apidhāya so 'yam avasāne
bandhyo 'vadhivāsara iva tuṣāradivasaḥ kadarthayati
surabhavane taruṇābhyāṃ parasparākṛṣṭadṛṣṭihṛdayābhyām
devārcanārtham udyatam anyonyasyārpitaṃ kusumam
sāyaṃ kuśeśayāntarmadhupānāṃ niryatāṃ nādaḥ
mitravyasanaviṣaṇṇaiḥ kamalair ākranda iva muktaḥ
sumahati manyunimitte mayaiva vihite'pi vepamānoruḥ
na sakhīnām api rudatī mamaiva vakṣaḥsthale patitā
subhaga vyajanavicālanaśithilabhujābhūd iyaṃ vayasyāpi
udvartanaṃ na sakhyāḥ samāpyate kiṃcid apagaccha
savrīḍā nakharadanārpaṇeṣu kupitā pragāḍham aciroḍhā
bahuyācñācaraṇagrahasādhyā roṣeṇa jāteyam
sugṛhītamalinapakṣā laghavaḥ parabhedinaḥ paraṃ tīkṣṇāḥ
puruṣā api viśikhā api guṇacyutāḥ kasya na bhayāya
svakapolena prakaṭīkṛtaṃ pramattatvakāraṇaṃ kim api
dviradasya durjanasya ca madaṃ cakāraiva dānam api
satyaṃ patir avidagdhaḥ sā tu svadhiyaiva nidhuvane nipuṇā
mārttikam ādhāya guruṃ dhanur adhigatam ekalavyena
saubhāgyamānavān sa tvayāvadhīryāpamānam ānītaḥ
svaṃ virahapāṇḍimānaṃ bhasmasnānopamaṃ tanute
sakhi mama karañjatailaṃ bahusaṃdeśaṃ praheṣyasīty uditā
śvaśuragṛhagamanamilitaṃ bāṣpajalaṃ saṃvṛṇoty asatī
saṃdarśayanti sundari kulaṭānāṃ tamasi vitatam avikalpe
maulimaṇidīpakalikā vartinibhā bhogino 'dhvānam
sarvaṃ vanaṃ tṛṇālyā pihitaṃ pītāḥ sitāṃśuravitārāḥ
pradhvaṃsāḥ panthāno malinenodgamya meghena
samyag aniṣpannaḥ san yo 'rthas tvarayā svayaṃ sphuṭīkriyate
sa vyaṅga eva bhavati prathamo vinatātanūja iva
sajjana eva hi vidyā śobhanāyai bhavati durjane moghā
na vidūradarśanatayā kaiścid upādīyate gṛdhraḥ
subhagaṃ vadati janas taṃ nijapatir iti naiṣa rocate mahyam
pīyūṣe'pi hi bheṣajabhāvopanate bhavaty aruciḥ
saudhagavākṣagatāpi hi dṛṣṭis taṃ sthitikṛtaprayatnam api
himagiriśikharaskhalitā gaṅgevairāvataṃ harati
sahagharmacāriṇī mama paricchadaḥ sutanu neha sandehaḥ
na tu sukhayati tuhinadinacchatracchāyeva sajantī
sakalaguṇaikaniketana dānavavāsena gharaṇiruharājaḥ
jāto 'si bhūtale tvaṃ satām anādeyaphalakusumaḥ
sundari tāṭaṅkamayaṃ cakram ivodvahati tāvake karṇe
nipatati nikāmatīkṣṇaṃ kaṭākṣabāṇo 'rjunapraṇayī
svādhīnaiva phalarddhir janopajīvyatvam ucchrayacchāyā
satpuṃso marubhūruha iva jīvanamātram āśāsyam
santāpamohakampān saṃpādayituṃ nihantum api jantūn
sakhi durjanasya bhūtiḥ prasarati dūraṃ jvarasyeva
sukhayitatarāṃ na rakṣati paricayaleśaṃ gaṇāṅganeva śrīḥ
kukakāminīva nojhati vāgdevī janmajanmāpi
svasadananikaṭe nalinīm abhinavajātacchadāṃ nirīkṣyaiva
hā gṛhiṇīti pralapaṃś cirāgataḥ sakhi patiḥ patitaḥ
saci caturānanabhāvād vaimukhyaṃ kvāpi naiva darśayati
ayam ekahṛdaya eva druhiṇa iva priyatamas tad api
satyaṃ madhuro niyataṃ vakro nūnaṃ kalādharo dayitaḥ
sa tu veda na dvitīyām akalaṅkaḥ pratipadindur iva
svasthānād api vicalati majjati jaladhau ca nīcam api bhajate
nijapakṣarakṣaṇamanāḥ sujano mainākaśaila iva
saṃvṛṇu bāṣpajalaṃ sakhi dṛśam uparajyāñjanena valayainām
dayitaḥ paśyatu pallavapaṅkajayor yugapad eva rucam
sā pāṇḍudurbalāṅgī nayasi tvaṃ yatra yāti tatraiva
kaṭhinīva kaitavavido hastagrahamātrasādhyā te
sakhi viśvagañjanīyā lakṣmīr iva kamalamukhi kadaryasya
tvaṃ pravayaso 'sya rakṣāvīkṣaṇamātropayogyāsi
hṛdayajñayā gavākṣe visadṛkṣaṃ kimapi kūjitaṃ sakhyā
yat kalahabhinnatalpā bhayakapaṭād eti māṃ sutanu
harati hṛdayaṃ śalākānihito 'ñjanatantur eṣa sakhi mugdhe
locanabāṇamucāntarbhrūdhanuṣā kiṇa ivollikhitaḥ
hasasi caraṇaprahāre talpād apasārito bhuvi svapiṣi
nāsadṛśe'pi kṛte priya mama hṛdayāt tvaṃ viniḥsarasi
hasati sapatnī śvaśrū roditi vadanaṃ ca pidadhate sakhyaḥ
svapnāyitena tasyāṃ subhaga tvannāma jalpantyām
hṛdayaṃ mama pratikṣaṇavihitāvṛttiḥ sakhe priyāśokaḥ
prabalo vidārayiṣyati jalakalaśaṃ nīralekheva
hanta virahaḥ samantāj jvalayati durvāratīvrasaṃvegaḥ
aruṇas tapanaśilām iva punar na māṃ bhasmatāṃ nayati
hṛtvā taṭini taraṅgair bhramitaś cakreṣu nāśaye nihitaḥ
phaladalavalkalarahitas tvayāntarikṣe tarus tyaktaḥ
hṛtakāñcivallibandhottarajaghanād aparabhogabhuktāyāḥ
ullasati romarājiḥ stanaśambhor garalalekheva
kṣīrasya tu dayitatvaṃ yato 'pi śāntopacāram āsādya
śailo 'ṅgāny ānamayati premṇaḥ śeṣo jvarasyeva
kṣāntam apasārito yac caraṇāvupadhāya supta evāsi
udghāṭayasi kim ūru niḥśvāsaiḥ pulakayann uṣṇaiḥ
kṣudrodbhavasya kaṭutāṃ prakaṭayato yacchataśca madam uccaiḥ
madhuno laghupuruṣasya ca garimā laghimā ca bhedāya
pūrvair vibhinnavṛttāṃ guṇāḍhyabhavabhūtibāṇaraghukāraiḥ
vāgdevīṃ bhajato mama santaḥ paśyantu ko doṣaḥ
satpātropanayocitasatpratibimbābhinavavastu
kasya na janayati harṣaṃ satkāvyaṃ madhuravacanaṃ ca
ekādhvaindvitīyātirbhuvanasārā sphuṭokticāturyā
pañceṣuṣaṭpadahitā bhūṣā śravaṇasya saptaśatī
kavisamarasiṃhanādaḥ svarānuvādaḥ sudhaikasaṃvādaḥ
vidvadvinodakandaḥ sandarbho 'yaṃ mayā sṛṣṭaḥ
udayanabalabhadrābhyāṃ saptaśatī śiṣyasodarābhyāṃ me
dyaur iva ravicandrābhyāṃ prakāśitā nirmalīkṛtya
haricaraṇāñjalim alaṃ kavivaraharṣāya buddhimān satatam
kṛtāryāsaptaśatīm etāṃ govardhanācāryaḥ
